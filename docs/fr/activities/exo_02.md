## Exercice
Nous allons représenter une formation de la manière suivante :
```Python
["Python, initiation", 2, 1200, 0]
```

Une formation est représenté par une liste où :
* Le premier élément est le titre de la formation
* Le second est sa durée
* Le troisième est son prix
* Le quatrième est un entier représentant le nombre d'inscrits. 

Cette valeur est déclarée et affectée à une variable dans le fichier `exo_02.py`.

* Affichez le titre de la formation.
* Affichez sa durée.

Voyez si vous pouvez avoir un code « compréhensible ». 

Enfin :
* affichez le nombre de participants
* incrémentez cette valeur de 1
* affichez le nombre de participants
* Incrémentez de nouveau plusieurs fois avec la même instruction.
